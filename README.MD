## 项目介绍

`Alice CodeCreator`是一个通用代码生成辅助工具，支持生成包括不限于html、js、java、xml等等各种类型代码。

- 按照数据库表结构和模板生成代码
- 支持多用户数据隔离
- 支持模板分组，每个分组可配置独立的参数属性
- 支持分享，用户可以分享自己的模板分组给其他用户使用
- 支持多数据源配置，动态加载数据源
- 支持不同数据库（TODO，目前仅支持MySQL，未来会支持Oracle和SQLServer）
